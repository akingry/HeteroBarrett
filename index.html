About the Mod

This mod provides a simple adjustment to the character Barrett in Bethesda's Starfield. He has a wife named Elsa instead of a husband named Ervin. The mod uses AI voices trained on the relevant characters and modifies the dialogue, terminal, quest, and player text. The pronouns she/her replace he/his.

"Modding is about player agency. This mod allows players to customize their universe according to their own gameplay preferences."

Installation

    Download the provided .7z archive
    Open Mod Organizer 2
    Click Install a new mod from an archive
    Select the .7z file
    Enable HeteroBarrett in the left pane
    Launch Starfield via MO2
