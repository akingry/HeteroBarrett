# HeteroBarrett

## Overview
This is a Bethesda Studios Starfield mod that makes the character Barrett into a heterosexual. He has a wife named Elsa instead of a husband named Ervin. The mod uses AI voices trained on the relevant characters and modifies the dialogue, terminal, quest, and player text. The pronouns she/her replace he/his.

## Installation
1. Download the provided `.7z` archive
2. Open **Mod Organizer 2**
3. Click **Install a new mod from an archive**
4. Select the `.7z` file
5. Enable **HeteroBarrett** in the left pane
6. Launch Starfield via MO2

## File Structure
The mod installs the following directories and files:

- `HeteroBarrett.esm`
- `Sound/`
  - `Voice/`
    - `Starfield.esm/`
      - `npcfajamamasa/`
      - `npcfandromedakepler/`
      - `npcfeilleyankton/`
      - `npcpaulinekeala/`
      - `npcsarahmorgan/`
      - `npcmbarrett/`
      - `npcmhelgihauksson/`
        - `*.wem`
- `Strings/`
  - `Starfield_en.DLSTRINGS`
  - `Starfield_en.ILSTRINGS`
  - `Starfield_en.STRINGS`
- `meta.ini`

The STRINGS files modify the in-game text, and the Voice folder contains seven audio subfolders for each character involved in quests related to Ervin (Barrett's husband).
